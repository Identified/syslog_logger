# syslog_logger
A Logger replacement that logs to syslog.  It is almost drop-in with a few caveats.

## Installation

Add this line to your application's Gemfile:

    gem 'syslog_logger'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install syslog_logger
